# MiroFish Full-Stack i18n — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add bilingual (English/Chinese) support with a UI toggle, locale-aware API messages, and locale-specific LLM prompts.

**Architecture:** Three layers — (1) vue-i18n on frontend with extracted message files, (2) thin backend message catalog for API responses, (3) prompt template files per locale for LLM services. Locale flows from frontend via Accept-Language header, captured in Flask middleware, passed explicitly to background threads.

**Tech Stack:** vue-i18n@9, Flask (existing), plain JSON + text files for translations.

**Spec:** `docs/superpowers/specs/2026-03-22-i18n-design.md`

- [ ] **Step 1: Create `backend/app/i18n/__init__.py`**

```python
import json
import os
from flask import g

_MESSAGES = {}
_DIR = os.path.dirname(__file__)

def _load_messages():
    if _MESSAGES:
        return
    for locale in ('en', 'zh-CN'):
        path = os.path.join(_DIR, f'{locale}.json')
        if os.path.exists(path):
            with open(path, 'r', encoding='utf-8') as f:
                _MESSAGES[locale] = json.load(f)

def msg(key: str, locale: str = None, **kwargs) -> str:
    _load_messages()
    locale = locale or getattr(g, 'locale', 'en')
    messages = _MESSAGES.get(locale, _MESSAGES.get('en', {}))
    template = messages.get(key, _MESSAGES.get('en', {}).get(key, key))
    return template.format_map(kwargs) if kwargs else template
```

```python
import os

_DIR = os.path.dirname(__file__)

def load_prompt(name: str, locale: str = 'en') -> str:
    path = os.path.join(_DIR, locale, f'{name}.txt')
    if not os.path.exists(path):
        path = os.path.join(_DIR, 'en', f'{name}.txt')
    with open(path, 'r', encoding='utf-8') as f:
        return f.read()
```

```js
import { createI18n } from 'vue-i18n'
import en from './en.json'
import zhCN from './zh-CN.json'

function detectLocale() {
  return localStorage.getItem('locale')
    || (navigator.language.startsWith('zh') ? 'zh-CN' : 'en')
}

const i18n = createI18n({
  legacy: false,
  locale: detectLocale(),
  fallbackLocale: 'en',
  messages: { en, 'zh-CN': zhCN }
})

export default i18n
```

```bash

import json
en = json.load(open('frontend/src/i18n/en.json'))
zh = json.load(open('frontend/src/i18n/zh-CN.json'))
def flat(d, prefix=''):
    r = {}
    for k, v in d.items():
        key = f'{prefix}.{k}' if prefix else k
        if isinstance(v, dict):
            r.update(flat(v, key))
        else:
            r[key] = v
    return r
en_keys = set(flat(en))
zh_keys = set(flat(zh))
missing = en_keys - zh_keys
if missing:
    print(f'Missing {len(missing)} keys in zh-CN.json:')
```
