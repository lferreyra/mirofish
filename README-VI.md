## ⚡ Tổng quan
 
**MiroFish** là một công cụ dự báo AI thế hệ mới được hỗ trợ bởi công nghệ đa tác tử (multi-agent). Bằng cách trích xuất thông tin gốc từ thế giới thực (như tin tức nóng hổi, dự thảo chính sách hoặc các tín hiệu tài chính), nó sẽ tự động xây dựng một thế giới kỹ thuật số song song với độ trung thực cao. Trong không gian này, hàng ngàn tác tử thông minh với tính cách độc lập, trí nhớ dài hạn và logic hành vi sẽ tự do tương tác và tiến hóa xã hội. Bạn có thể chèn các biến số một cách linh hoạt từ "Góc nhìn của Chúa" để suy luận chính xác các quỹ đạo tương lai — **diễn tập tương lai trong một sa bàn kỹ thuật số, và giành chiến thắng trong các quyết định sau vô số lần mô phỏng**.
 
> Bạn chỉ cần: Tải lên các tài liệu gốc (báo cáo phân tích dữ liệu hoặc các câu chuyện tiểu thuyết thú vị) và mô tả yêu cầu dự báo của bạn bằng ngôn ngữ tự nhiên</br>
> MiroFish sẽ trả về: Một báo cáo dự báo chi tiết và một thế giới kỹ thuật số có độ trung thực cao với khả năng tương tác sâu sắc
 
### Tầm nhìn của chúng tôi
 
MiroFish tận tâm tạo ra một tấm gương trí tuệ bầy đàn phản ánh thực tế. Bằng cách nắm bắt sự trỗi dậy của tập thể được kích hoạt bởi các tương tác cá nhân, chúng tôi phá vỡ các giới hạn của dự báo truyền thống:
 
- **Ở cấp độ Vĩ mô**: Chúng tôi là một phòng thí nghiệm diễn tập cho các nhà ra quyết định, cho phép các chính sách và quan hệ công chúng được thử nghiệm với rủi ro bằng không.
- **Ở cấp độ Vi mô**: Chúng tôi là một sa bàn sáng tạo cho người dùng cá nhân — cho dù là suy luận kết thúc tiểu thuyết hay khám phá các kịch bản giàu trí tưởng tượng, mọi thứ đều có thể thú vị, vui nhộn và dễ tiếp cận.
 
Từ những dự báo nghiêm túc đến những mô phỏng vui nhộn, chúng tôi để mọi câu hỏi "nếu như" chứng kiến kết quả của nó, làm cho việc dự báo vạn vật trở nên khả thi.

## 🔄 Quy trình làm việc
 
1. **Xây dựng Đồ thị**: Trích xuất dữ liệu gốc & Tiêm trí nhớ cá nhân/tập thể & Xây dựng GraphRAG.
2. **Thiết lập Môi trường**: Trích xuất mối quan hệ thực thể & Tạo nhân vật (persona) & Tiêm cấu hình tác tử.
3. **Mô phỏng**: Mô phỏng song song trên hai nền tảng & Tự động phân tích yêu cầu dự báo & Cập nhật trí nhớ thời gian động.
4. **Tạo Báo cáo**: ReportAgent với bộ công cụ phong phú để tương tác sâu với môi trường sau mô phỏng.
5. **Tương tác Sâu**: Trò chuyện với bất kỳ tác tử nào trong thế giới mô phỏng & Tương tác với ReportAgent.
 
## 🚀 Bắt đầu nhanh
 
### Lựa chọn 1: Triển khai từ Mã nguồn (Khuyến nghị)
 
#### Điều kiện tiên quyết
 
| Công cụ | Phiên bản | Mô tả | Kiểm tra cài đặt |
|------|---------|-------------|-------------------|
| **Node.js** | 18+ | Môi trường chạy Frontend, bao gồm npm | `node -v` |
| **Python** | ≥3.11, ≤3.12 | Môi trường chạy Backend | `python --version` |
| **uv** | Mới nhất | Trình quản lý gói Python | `uv --version` |
 
#### 1. Cấu hình biến môi trường
 
```bash
# Sao chép tệp cấu hình ví dụ
cp .env.example .env
 
# Chỉnh sửa tệp .env và điền các API key cần thiết
```
 
**Các biến môi trường bắt buộc:**
 
```env
# Cấu hình LLM API (hỗ trợ bất kỳ LLM API nào có định dạng OpenAI SDK)
# Khuyến nghị: Mô hình Qwen-plus của Alibaba thông qua Nền tảng Bailian: https://bailian.console.aliyun.com/
# Tiêu tốn khá nhiều tài nguyên, hãy thử mô phỏng với ít hơn 40 hiệp trước
LLM_API_KEY=your_api_key
LLM_BASE_URL=https://dashscope.aliyuncs.com/compatible-mode/v1
LLM_MODEL_NAME=qwen-plus
 
# Cấu hình Zep Cloud
# Hạn ngạch miễn phí hàng tháng là đủ cho nhu cầu sử dụng đơn giản: https://app.getzep.com/
ZEP_API_KEY=your_zep_api_key
```
 
#### 2. Cài đặt Phụ thuộc
 
```bash
# Cài đặt tất cả các phụ thuộc chỉ với một cú nhấp chuột (gốc + frontend + backend)
npm run setup:all
```
 
Hoặc cài đặt từng bước:
 
```bash
# Cài đặt phụ thuộc Node (gốc + frontend)
npm run setup
 
# Cài đặt phụ thuộc Python (backend, tự động tạo môi trường ảo)
npm run setup:backend
```
 
#### 3. Khởi chạy Dịch vụ
 
```bash
# Khởi chạy cả frontend và backend (chạy từ thư mục gốc dự án)
npm run dev
```
 
**URL Dịch vụ:**
- Frontend: `http://localhost:3000`
- Backend API: `http://localhost:5001`
 
**Khởi chạy riêng lẻ:**
 
```bash
npm run backend   # Chỉ khởi chạy backend
npm run frontend  # Chỉ khởi chạy frontend
```
 
### Lựa chọn 2: Triển khai với Docker
 
```bash
# 1. Cấu hình biến môi trường (tương tự như triển khai mã nguồn)
cp .env.example .env
 
# 2. Kéo image và khởi chạy
docker compose up -d
```
 
Mặc định sẽ đọc `.env` từ thư mục gốc, ánh xạ các cổng `3000 (frontend) / 5001 (backend)`
 
> Địa chỉ mirror để kéo image nhanh hơn được cung cấp dưới dạng chú thích trong `docker-compose.yml`, hãy thay thế nếu cần thiết.
